# Assignment 2 - Task 2
# Program to find the sum of integers from 1 to 50

# Initialize a variable for sum
total = 0

# Loop from 1 to 50
for i in range(1, 51):
    total = total + i

# Display the result
print("The sum of numbers from 1 to 50 is:", total)

output:
The sum of numbers from 1 to 50 is: 1275.
